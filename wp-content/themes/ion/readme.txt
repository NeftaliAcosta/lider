== Description ==
Description: The Ion theme is a theme specifically designed to be used in a native mobile app.

== Installation ==

1. Unzip the ion.x.x.x.zip file. Be sure that the folder is named ion for updates.
2. Upload the ion folder using your preferred FTP software to wp-content/themes/
3. Go to the AppPresser menu.
4. From the "App-only theme" downdown select Ion.
5. Press the "Save Settings" button on the bottom of the page.
6. Press the "Open Customer" to customizer your new theme.
