== Description ==
Easily add a touch-enabled slider and carousel to your app.  This is incredibly useful for apps that display a lot of photos, for example an ECommerce application or a media-rich news organization or community.

== Installation ==

1. Upload AppSwiper to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Press the customer button in in your AppPresser settings to edit your mobile front page or use a shortcode to add it to any other page.
